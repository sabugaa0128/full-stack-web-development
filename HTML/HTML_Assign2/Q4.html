HTML Tags List

  1. <!DOCTYPE HTML>: Document Type Declaration

    It declares the version of HTML

  2. <html>: Root Element

    Encloses the entire HTML document.

  3. <head>: Document Head

    Contains meta-information about the HTML document, such as title, meta tags, and linked stylesheets.

  4. <title>: Document Title

    Defines the title of the HTML document, displayed in the browser's title bar or tab.

  5. <body>: Document Body

    Contains the content of the HTML document, including text, images, links, and other elements.

  6. <h1> to <h6>: Heading Elements

    Define headings of various levels, where <h1> is the highest and <h6> is the lowest.

  7. <p>: Paragraph

    Defines a paragraph of text.

  8. <a>: Anchor

    Creates hyperlinks to other web pages or resources.

  9. <img>: Image

    Embeds an image in the HTML document.

  10. <ul>: Unordered List

    Defines an unordered list using bullet points.

  11. <ol>: Ordered List

    Defines an ordered list using numbers or other ordering styles.

  12. <li>: List Item

    Defines an item in a list (either ordered or unordered).

  13. <div>: Division

    Groups content for styling or scripting purposes.

  14. <span>: Inline Element

    Defines a small section of text with its own styling or scripting.

  15. <form>: Form

    Encloses form elements for user input, such as text fields, buttons, and checkboxes.

  16. <input>: Input

    Defines an input field within a form, like text input, radio buttons, or checkboxes.

  17. <textarea>: Textarea

    Defines a multiline text input field within a form.

  18. <button>: Button

    Defines a clickable button within a form.

  19. <select>: Select

    Creates a dropdown list within a form.

  20. <option>: Option

    Defines an option within a dropdown list.
