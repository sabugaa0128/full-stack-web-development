HTML5 Features Showcase

Feature 1: Semantic Elements

HTML5 introduces semantic elements that provide more meaning to the structure of a webpage. Some key semantic elements include:

  • <header>
  • <nav>
  • <main>
  • <article>
  • <section>
  • <aside>
  • <footer>

These elements improve the document's structure and make it more accessible and readable.

Feature 2: Audio and Video Support

HTML5 provides native support for embedding audio and video content using the <audio> and <video> elements:

        
          <audio controls>
            <source src="audio.mp3" type="audio/mp3">
            Your browser does not support the audio tag.
          </audio>

          <video width="320" height="240" controls>
            <source src="video.mp4" type="video/mp4">
            Your browser does not support the video tag.
          </video>

These elements allow seamless integration of multimedia content.

Feature 3: Form Enhancements

HTML5 introduces new form elements and attributes for improved user experience:

  • <input type="email"> for email input.
  • <input type="url"> for URL input.
  • <input type="tel"> for telephone input.
  • <input type="number"> for numeric input.
  • <input type="date"> for date input.
  • <input type="color"> for color selection.

These enhancements make form validation and user interaction more robust.
